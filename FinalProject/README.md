# Medicare+ - A Medical Storefront Web Application

Medicare+ is a multi-page JavaScript web application that simulates a simple online store for medical products.  
It includes dynamic product rendering, filtering, cart management using localStorage, and a validated contact form.

2. Right-click `index.html`.
3. Choose **Open with Live Server**.
